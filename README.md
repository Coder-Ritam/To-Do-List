# Simple To-Do List 

A clean and functional To-Do List application built to practice modern JavaScript concepts.

## Features
* Add Tasks: Type a task and hit "Enter" or click the button.
* Edit Tasks: Click "Edit" to modify text directly inline.
* Data Persistence: Uses `localStorage` so tasks survive a page refresh.
* Task Management: Delete tasks or mark them as completed.
* Responsive Design: Looks good on mobile and desktop.

## Built With
* HTML5
* CSS3
* JavaScript (Vanilla)
